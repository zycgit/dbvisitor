# Mapper Interface (BaseMapper)

`BaseMapper<T>` is a generic data access interface provided by dbVisitor. Based on [Object Mapping](../mapping/about), it automatically generates CRUD statements **without writing any SQL**.

```java title='Quick Start'
BaseMapper<User> mapper = session.createBaseMapper(User.class);

// Insert
mapper.insert(user);

// Query by primary key
User found = mapper.selectById(1L);

// Pagination query
Page page = PageObject.of(0, 20);
PageResult<User> result = mapper.pageBySample(null, page);
```

:::tip[Tip]
How to obtain a Session depends on your project architecture. See **[Framework Integration](../../yourproject/buildtools#integration)** for details.
:::
