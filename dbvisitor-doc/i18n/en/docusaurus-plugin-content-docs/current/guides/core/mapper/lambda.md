# Use the Fluent API

The zero-argument `insert`, `update`, `delete`, and `query` on BaseMapper let you call the [Fluent API](../lambda/about#principle) directly, without preparing data first.

```java title='Example: insert'
User user = ...

BaseMapper<User> mapper = session.createBaseMapper(User.class);
mapper.insert().applyEntity(user)
               .executeSumResult();
```

```java title='Example: update'
BaseMapper<User> mapper = session.createBaseMapper(User.class);
mapper.update().eq(User::getId, 1)              // 匹配条件
               .updateTo(User::getName, "Mary") // Update field via lambda
               .updateTo(User::getStatus, 2)    // Chain to update multiple fields
               .doUpdate();
```

```java title='Example: delete'
BaseMapper<User> mapper = session.createBaseMapper(User.class);
mapper.delete().eq(User::getId, 1) // Match condition
               .doDelete();
```

```java title='Example: query'
BaseMapper<User> mapper = session.createBaseMapper(User.class);
List<User> result = null;
result = mapper.query().le(User::getId, 100)   // ID <= 100
                       .queryForList();        // Map rows to entity
```
