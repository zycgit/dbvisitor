# Dependencies

All dbVisitor modules and their dependencies are available on [Maven Central](https://central.sonatype.com/search?q=dbvisitor).
Any project that uses a dependency manager (Apache Maven, Gradle, sbt, leiningen, Apache Ivy, etc.) can pull them in.

Latest release: **<Highlight color="rgb(227 17 108)">{Vars.lastReleaseVer}</Highlight>**

## Apache Maven

Add dbVisitor to a Maven project:

```xml
<dependency>
    <groupId>net.hasor</groupId>
    <artifactId>dbvisitor</artifactId>
    <version>latest version</version>
</dependency>
```

## Gradle

  Add dbVisitor to a Gradle project:

```text
// Gradle
implementation 'net.hasor:dbvisitor:latest-version'
```

## Mixing dbVisitor versions

dbVisitor does not guarantee compatibility across different versions (for example, 5.3.1 with 6.0.0).
Use the same dbVisitor version across all components in a project or service.
