## How to use

To use dbVisitor in a plain project, just provide a `DataSource` or `Connection`.

dbVisitor APIs you can choose from:
- Use raw SQL via the [Programmatic API](../api/program_api).
- Use annotations on interfaces via the [Declarative API](../api/declarative_api).
- Use the unified [Fluent API](../api/lambda_api) across supported databases.
- Use predefined methods in [BaseMapper](../api/base_mapper) for common operations.
- Use [Mapper XML](../api/file_mapper) files to centrally manage SQL.
