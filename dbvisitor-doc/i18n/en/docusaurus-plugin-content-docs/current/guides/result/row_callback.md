# RowCallbackHandler

RowCallbackHandler processes each record instead of fetching and storing them. It is useful for streaming large datasets without leaving data in memory.

```java title='Example'
public class UserRowCallBackHandler implements RowCallBackHandler {
    @Override
    public void processRow(ResultSet rs, int rowNum) throws SQLException {
        ...
    }
}
```

## How to Use

```java title='Example: Programmatic API'
UserRowCallBackHandler userCallBackHandler = ...
List<User> result = jdbc.queryForList("select * from users", userCallBackHandler);
```

```java title='Example: Declarative API'
@SimpleMapper
public interface UserMapper {
    @Query(value = "select * from users where id > #{id}",
           resultRowCallback = UserRowCallBackHandler.class)
    void listUsers(@Param("id") long searchId);
}
```

```java title='Example: Builder'
List<UserVO> result = adapter.queryByEntity(User.class)
                             .le(User::getId, 100)              // match IDs <= 100
                             .queryForList(userCallBackHandler);// process via callback
```
