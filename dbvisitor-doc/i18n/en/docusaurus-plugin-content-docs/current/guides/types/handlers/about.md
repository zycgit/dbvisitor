## Basic Type handlers

dbVisitor ships many handlers. Naming follows three patterns:

1. `<JavaTypeName>TypeHandler` or `<JdbcTypeName>TypeHandler`
2. `<JdbcTypeName>As<JavaTypeName>TypeHandler` or `<OneType>As<AnotherType>TypeHandler`
3. `<xxx>TypeHandler`
