# String/char type handlers

Handlers are in `net.hasor.dbvisitor.types.handler.string`.

## Common handlers

| Handler                      | Java type          | Purpose                             |
|--------------------------|------------------|-----------------------------------|
| StringTypeHandler        | java.lang.String | Use getString/setString for strings |
| ClobAsStringTypeHandler  | java.lang.String | Use getClob/setClob for strings     |
| NStringTypeHandler       | java.lang.String | Use getNString/setNString for strings |
| NClobAsStringTypeHandler | java.lang.String | Use getNClob/setNClob for strings   |

## Character types

| Handler                      | Java type             | Purpose                                      |
|--------------------------|---------------------|--------------------------------------------|
| StringAsCharTypeHandler  | java.lang.Character | getString/setString; uses the first char only |
| NStringAsCharTypeHandler | java.lang.Character | getNString/setNString; uses the first char only |

## Type conversions

| Handler                    | Java type      | Purpose          |
|------------------------|--------------|----------------|
| StringAsUrlTypeHandler | java.net.URL | Read/write URL values |
| StringAsUriTypeHandler | java.net.URI | Read/write URI values |

## Special support

| Handler               | Java type          | Purpose                                |
|-------------------|------------------|--------------------------------------|
| EnumTypeHandler   | java.lang.Enum   | Enum support; see [enum handler](../enum-handler) |
| SqlXmlTypeHandler | java.lang.String | Use getSQLXML/setSQLXML for XML strings |
