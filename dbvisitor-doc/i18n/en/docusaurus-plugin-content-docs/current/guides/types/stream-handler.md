# Stream Type Handlers

Stream type handlers are located in the `net.hasor.dbvisitor.types.handler.io` package.

## InputStream Handlers

| Handler | Purpose |
|---|---|
| `BlobAsInputStreamTypeHandler` | Reads/writes BLOB data as a stream via getBlob/setBlob |
| `BytesAsInputStreamTypeHandler` | Copies stream data to memory for reading/writing via getBytes/setBytes |
| `SqlXmlAsInputStreamTypeHandler` | Reads/writes data via getSQLXML/setSQLXML using binaryStream |

## Reader Handlers

| Handler | Purpose |
|---|---|
| `ClobAsReaderTypeHandler` | Reads/writes CLOB data as a stream via getClob/setClob |
| `NClobAsReaderTypeHandler` | Reads/writes NCLOB data as a stream via getNClob/setNClob |
| `NStringAsReaderTypeHandler` | Copies stream data to memory for reading/writing via getNString/setNString |
| `SqlXmlAsReaderTypeHandler` | Reads/writes data via getSQLXML/setSQLXML using characterStream |
| `StringAsReaderTypeHandler` | Copies stream data to memory for reading/writing via getString/setString |
