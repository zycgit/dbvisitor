# Array type handlers

Array handlers live in `net.hasor.dbvisitor.types.handler.array`.

| Handler            | Java type       | Purpose                                  |
|--------------------|-----------------|------------------------------------------|
| ArrayTypeHandler   | java.sql.Array  | Read/write arrays via getArray/setArray  |
| PgArrayTypeHandler | java.sql.Array  | Read/write PostgreSQL arrays             |


When creating a `java.sql.Array` you must provide the element type:

```java
Array array = ps.getConnection().createArrayOf(typeName, ...);
```
