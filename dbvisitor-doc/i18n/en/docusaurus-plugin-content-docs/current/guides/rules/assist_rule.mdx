# Assist Rules

| Rule                                         | Description |
|----------------------------------------------|-------------|
| [RESULT rule](./assist_rule#result-set)      | No effect in the query text but influences how result sets are fetched in specific scenarios. |
| [DEFAULTRESULT rule](./assist_rule#default)  | When multi-result or stored procedures return multiple result sets, use defaultResult to avoid configuring multiple RESULT rules. |

## RESULT Rule {#result-set}

RESULT has no effect on the query text but influences result handling in specific scenarios. Usage:

```text title='Usage'
@{resultSet, key1 = value1, key2 = value2}
```

Supported scenarios:
- [Stored procedure calls](../core/jdbc/procedure)
- [Multi-result](../core/jdbc/multi)
- Using `PreparedMultipleResultSetExtractor` or `CallableMultipleResultSetExtractor` to receive results.

Attributes:

| Name       | Type                | Description |
|------------|---------------------|-------------|
| name       | String              | <TagGray/> Name of the OUT argument in the result set. When mode is `INOUT`, this can distinguish the names for input vs output. |
| javaType   | Class               | <TagGray/> Map cursor result sets to specific Java types; see [Object Mapping](../core/mapping/about). |
| rowMapper  | RowMapper           | <TagGray/> Same as above; see [RowMapper](../result/for_mapper). |
| rowHandler | RowCallbackHandler  | <TagGray/> Same as above; see [RowCallbackHandler](../result/row_callback). |
| extractor  | ResultSetExtractor  | <TagGray/> Same as above; see [ResultSetExtractor](../result/for_extractor). |

## DEFAULTRESULT Rule {#default}

DEFAULTRESULT also has no text effect but influences result handling in specific scenarios. Usage:

```text title='Usage'
@{defaultResult, key1 = value1, key2 = value2}
```

Supported scenarios:
- [Stored procedure calls](../core/jdbc/procedure)
- [Multi ResultSet](../core/jdbc/multi)
- Using `PreparedMultipleResultSetExtractor` or `CallableMultipleResultSetExtractor` to receive results.

Attributes:

| Name       | Type                | Description |
|------------|---------------------|-------------|
| javaType   | Class               | <TagGray/> Map cursor result sets to specific Java types; see [Object Mapping](../core/mapping/about). |
| rowMapper  | RowMapper           | <TagGray/> Same as above; see [RowMapper](../result/for_mapper). |
| rowHandler | RowCallbackHandler  | <TagGray/> Same as above; see [RowCallbackHandler](../result/row_callback). |
| extractor  | ResultSetExtractor  | <TagGray/> Same as above; see [ResultSetExtractor](../result/for_extractor). |
