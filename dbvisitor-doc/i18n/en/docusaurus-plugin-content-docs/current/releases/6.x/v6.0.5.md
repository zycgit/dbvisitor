# v6.0.5 (2025-07-11)

```xml
<dependency>
    <groupId>net.hasor</groupId>
    <artifactId>dbvisitor</artifactId>
    <version>6.0.5</version>
</dependency>
```

## Impact

- MapperRegistry and MappingRegistry

## Changes

- [Added] `MapperRegistry.loadMapper` gains overloads accepting `InputStream` and `Document`.
- [Added] `MappingRegistry.loadMapping` gains overloads accepting `InputStream` and `Document`.
- [Added] `MapperRegistry` adds protected `removeSqlConfig` so extensions can remove SqlConfig dynamically.
- [Improved] `MapperRegistry` methods `testMapper`, `matchType`, `matchMethod`, `getMethodDynamicResolve`, `getXmlDynamicResolve`, `applyResultConfig` are now protected for extension.
- [Improved] Extract `addSqlConfig` from `tryLoadNode`, make it protected to allow dynamic SqlConfig additions in extensions.
