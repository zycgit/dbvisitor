# v6.0.1 (2025-04-06)

```xml
<dependency>
    <groupId>net.hasor</groupId>
    <artifactId>dbvisitor</artifactId>
    <version>6.0.1</version>
</dependency>
```

## Changes

- [Improved] WrapperAdapter renamed back to LambdaTemplate in 6.0.1 for higher acceptance.
- [Improved] Solon plugin package renamed to `dbvisitor-solon-plugin` per Solon norms.
- [Improved] When Solon plugin lacks dialect config, pagination threw null exceptions; now still throws but via `DefaultSqlDialect` as `UnsupportedOperationException`.
- [Improved] Solon plugin injected types without dbVisitor config previously caused dbVisitor exceptions; now dbVisitor no-ops instead.
- [Improved] Corrected inaccurate website content.
