# v0.0.12 (2015-05-09)

```xml
<dependency>
    <groupId>net.hasor</groupId>
    <artifactId>hasor-core</artifactId>
    <version>0.0.12</version>
</dependency>
```

## Added
- ORM toolkit with a very lightweight database operation framework.
- `SimpleDBModule` utility to configure a transaction manager for a data source.
- Support for seven transaction propagation behaviors.
- Support for isolation level configuration.
- Support for multiple data sources.
- Support for multiâ€“data source transaction control (non-JPA).
- Simple declarative transaction annotation `@Transactional`.
- Developers can customize transaction interceptors and annotations.
- `SavepointDataSourceHelper` interface added and supported by `DefaultDataSourceHelper`.
- Methods to obtain the current connection via `DataSourceUtils`.
- `net.hasor.db.jdbc` package provides a lightweight JDBC API for Java applications.
- `net.hasor.db.datasource` package provides data source services for Java applications.
- Added transaction manager and transaction interceptor.
- `LocalDataSourceHelper` supports binding multiple database connections per thread.
