# v5.1.0 (2022-08-01)

```xml
<dependency>
    <groupId>net.hasor</groupId>
    <artifactId>dbvisitor</artifactId>
    <version>5.1.0</version>
</dependency>
```

## Added
- `dbvisitor-faker` tool (test data generation), v5.1.0 preview.
- `dbvisitor-hasor` integration (Hasor 2.4.5).
- `dbvisitor-guice` integration (Guice 5.1.0).
- Table adds `catalog` dimension.
- `ConditionSqlDialect` adds `randomQuery` for random selection dialects.

## Improved
- Provider utilities used only by Hasor moved into the Hasor project.
- SpringBoot integration improved for IDE property recognition.
- Pagination params `start`, `limit` changed to `long`.

## Fixed
- SqlServer special table name formatting.
