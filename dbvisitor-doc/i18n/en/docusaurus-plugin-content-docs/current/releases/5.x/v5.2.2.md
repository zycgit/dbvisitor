# v5.2.2 (2023-02-24)

```xml
<dependency>
    <groupId>net.hasor</groupId>
    <artifactId>dbvisitor</artifactId>
    <version>5.2.2</version>
</dependency>
```

## Added
- `@RefMapper` can omit `value`; defaults to class path + name as XML path.
- TypeHandlers for PG array and Money types.
- TypeHandlers storing `BigDecimal`/`BigInteger` as String.
- TypeHandler storing `LocalDateTime` as `java.sql.Timestamp`.
- Faker `dbType\customTpcConf` for custom tpc config files.
- Faker DSL-based `TypeProcessorFactory`; MySQL/PG/Oracle/SQLServer implementations switched to DSL.

## Improved
- `XmlTableMappingResolve` reduces stack depth in exceptions.
- Dependencies: cobble 4.5.3, ognl 3.3.4.
- TypeHandler class names adjusted to new naming rules.

## Fixed
- `META-INF/custom.keywords` loader only recognizing one entry.
