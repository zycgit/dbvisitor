# v5.0.2 (2022-07-02)

```xml
<dependency>
    <groupId>net.hasor</groupId>
    <artifactId>dbvisitor</artifactId>
    <version>5.0.2</version>
</dependency>
```

## Added
- `TransactionHelper` utility.
- `mapperDisabled` parameter: with SpringBoot integration, load mapper XML without registering beans.
- `queryForIterator` method.
- Custom keywords via `META-INF/custom.keywords`; SQL uses quotes when encountering them.
- Mapper supports interface default methods.

## Improved
- Upgrade cobble to 4.4.0.
- Remove `@Transactional` annotation (to be provided via plugin later).

## Fixed
- Incorrect DTD usage in example project.
