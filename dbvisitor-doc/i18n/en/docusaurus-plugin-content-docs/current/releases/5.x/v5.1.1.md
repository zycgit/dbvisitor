# v5.1.1 (2022-09-29)

```xml
<dependency>
    <groupId>net.hasor</groupId>
    <artifactId>dbvisitor</artifactId>
    <version>5.1.1</version>
</dependency>
```

## Added
- dbvisitor adds four new properties: `insertTemplate`, `setValueTemplate`, `whereColTemplate`, `whereValueTemplate` for deeper SQL customization.
- `SqlDialect` adds `fmtName`.
- Three GIS TypeHandlers based on jts.

## Improved
- Dialect registration for Oracle and SqlServer.
- Faker reverse-lookup capability.
- Faker supports full-type data generation for MySQL/PG/SQL Server.
