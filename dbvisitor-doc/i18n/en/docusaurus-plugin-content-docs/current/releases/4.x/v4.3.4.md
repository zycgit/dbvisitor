# v4.3.4 (2022-04-29)

```xml
<dependency>
    <groupId>net.hasor</groupId>
    <artifactId>dbvisitor</artifactId>
    <version>4.3.4</version>
</dependency>
```

## Added
- `DynamicConnection` adds `releaseConnection` for external datasource managers to release connections.
- `DalRegistry.findTableMapping` overload to find by `Class`.
- `MapperScanner` utility.
- UUID-based primary key generator.

## Improved
- Spring XML configuration supported for dbVisitor.
- Both `@RefMapper` and `@SimpleMapper` are annotated with `@DalMapper`.
