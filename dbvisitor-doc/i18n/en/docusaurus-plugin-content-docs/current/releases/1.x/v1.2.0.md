# v1.2.0 (2015-08-20)

```xml
<dependency>
    <groupId>net.hasor</groupId>
    <artifactId>hasor-core</artifactId>
    <version>1.2.0</version>
</dependency>
```

## Added
- `net.hasor.db.transaction.Manager` adds a callback-style transaction execution method.

## Improved
- Transaction interceptors are split into a standalone transaction plugin, no longer part of DB.
