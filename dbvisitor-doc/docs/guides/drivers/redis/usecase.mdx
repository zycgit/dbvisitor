### 引入依赖

## JDBC 连接

使用标准的 JDBC URL 格式进行连接。

```text
jdbc:dbvisitor:jedis://<host>:<port>?database=<database>&<parameters>
```

- `host`: Redis 服务器地址。
- `port`: Redis 服务器端口（默认 6379）。
- `database`: 数据库索引（默认 0）。
- `parameters`: 连接参数，详见 [连接参数](./params)。

### 示例代码

```java
public class RedisJdbcExample {
    public static void main(String[] args) throws Exception {
        String url = "jdbc:dbvisitor:jedis://127.0.0.1:6379/0";
        try (Connection conn = DriverManager.getConnection(url)) {
            // Statement Example
            try (Statement stmt = conn.createStatement()) {
                try (ResultSet rs = stmt.executeQuery("GET mykey")) {
                    while (rs.next()) {
                        System.out.println(rs.getString(1));
                    }
                }
            }

            // PreparedStatement Example
            try (PreparedStatement pstmt = conn.prepareStatement("SET ? ?")) {
                pstmt.setString(1, "acc");
                pstmt.setString(2, "123");
                pstmt.executeUpdate();
            }
        }
    }
}
```
